# Prompt 20 - Pruebas de Estres y Reporte

```text
Usando el prompt principal como contexto, prepara pruebas de estres y su documentacion.

Debes crear:
- Script o configuracion para pruebas de carga.
- Escenarios sobre login, listado de partidos, predicciones, rankings y podios.
- Escenario controlado para recalculo o sincronizacion.
- docs/stress-testing.md con metodologia y resultados esperados.

Reglas:
- No saturar servicios externos reales.
- Mockear o limitar llamadas a Football-Data.org.
- Documentar metricas: usuarios concurrentes, latencia, errores y throughput.

Al finalizar:
- Indica como ejecutar las pruebas.
- Actualiza doc/progreso-prompts.md marcando Prompt 20 como completado.
```
