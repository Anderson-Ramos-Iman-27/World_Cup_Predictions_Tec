# Prompt 01 - Crear Estructura Base del Monorepo

```text
Usando el prompt principal como contexto, crea la estructura base del monorepo para el proyecto.

Debes crear:
- apps/frontend
- apps/backend
- infra/nginx
- infra/docker/postgres
- infra/docker/redis
- docs
- README.md
- .env.example inicial
- docker-compose.yml inicial
- docker-compose.prod.yml inicial vacio o preparado

No implementes todavia logica de negocio.

Incluye scripts base y estructura pensada para:
- Next.js App Router en frontend.
- NestJS en backend.
- Docker y Docker Compose.
- Documentacion en docs.

Al finalizar:
- Explica la estructura creada.
- Indica como continuar con el Prompt 02.
- Actualiza doc/progreso-prompts.md marcando Prompt 01 como completado.
```
