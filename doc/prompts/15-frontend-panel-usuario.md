# Prompt 15 - Frontend: Panel de Usuario

```text
Usando el prompt principal como contexto, implementa el panel de usuario.

Pantallas:
- Dashboard de usuario.
- Listado de partidos.
- Detalle de partido.
- Formulario de prediccion.
- Mis predicciones.
- Ranking global.
- Perfil de usuario.

Reglas:
- Usar SSR para vistas que convenga cargar rapido.
- Usar CSR para formularios e interacciones.
- Validar formularios antes de enviar.
- Mostrar mensajes claros cuando el partido ya no permita predicciones.

Al finalizar:
- Prueba el flujo usuario: ver partidos, predecir y consultar ranking.
- Actualiza doc/progreso-prompts.md marcando Prompt 15 como completado.
```
