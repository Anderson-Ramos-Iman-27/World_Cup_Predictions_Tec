# Prompt 03 - Configurar Frontend Next.js Base

```text
Usando el prompt principal como contexto, configura el frontend base en apps/frontend.

Debes implementar:
- Next.js con App Router.
- TypeScript.
- Tailwind CSS.
- Layout principal.
- Pagina inicial simple de la plataforma.
- Estructura src/app, components, features, hooks, lib, styles y types.
- Cliente HTTP base apuntando al backend mediante variable de entorno.
- Estilos globales.

No implementes aun pantallas completas de usuario o administrador.

Al finalizar:
- Ejecuta build o lint si esta disponible.
- Explica como levantar el frontend.
- Actualiza doc/progreso-prompts.md marcando Prompt 03 como completado.
```
