# Prompt 16 - Frontend: Salas Personalizables y Podios

```text
Usando el prompt principal como contexto, implementa en frontend el modulo de salas.

Pantallas y componentes:
- Mis salas.
- Crear sala.
- Editar sala propia.
- Unirse a sala por codigo o invitacion.
- Detalle de sala.
- Integrantes de sala.
- Podio por sala.
- Card o box de sala con color personalizable.

Reglas:
- El creador de la sala puede editar nombre, descripcion y color.
- Un usuario no propietario no debe ver acciones de edicion, salvo ADMIN si aplica.
- Cada sala debe mostrar integrantes y podio.
- El color elegido debe verse en el box/card de la sala.
- Usar Tailwind CSS y componentes responsive.

Al finalizar:
- Verifica creacion, edicion, union y podio.
- Actualiza doc/progreso-prompts.md marcando Prompt 16 como completado.
```
